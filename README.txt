
# MAKE GIOVI DOORS

## Istruzioni

1. Inserisci i tuoi file SVG all'interno della cartella `/svg/{modello}/{motore}.svg`
   - Esempio: `svg/active-roll/frontale-destro.svg`
2. All'interno del file SVG, usa questi segnaposto:
   - {{LARGHEZZA_MAX}}
   - {{ALTEZZA_MAX}}
   - {{DIM_MONTANTI}}
   - {{LUNGHEZZA_MONTANTE}}

3. Apri `index.html` nel browser per usare il configuratore.

## Pubblicazione

Puoi caricare il contenuto della cartella su GitHub Pages per pubblicarlo online.
